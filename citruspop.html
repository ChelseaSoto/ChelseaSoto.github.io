About Citrus POP!

Citronaut is on the beach planting citrobombs to sandcastles and non-UCF-affiliated Knights who are trying to ruin his fun!

Citrus POP! is based on Bomberman with a University of Central Florida (UCF) theme.

This is the first project for my game production class. We were required to recreate a classic arcade game and add a UCF theme to it. Each person needed to create at lease one art asset and we were all learning how to use GitHub Desktop as version control while using Unity Engine.
